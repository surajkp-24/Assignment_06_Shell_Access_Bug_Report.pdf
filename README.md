# Assignment 06 – Shell Access Bug Report

## 📌 Project Overview
This project demonstrates vulnerability assessment and security reporting in a controlled Metasploitable 2 lab environment. The assessment focuses on identifying a vulnerable service that allows unauthorized shell access.

---

## 🎯 Objective
The objective of this assignment is to:
- Identify a vulnerable service
- Verify shell access in a safe lab environment
- Document the vulnerability professionally
- Analyze the security impact
- Recommend remediation steps

---

## 🖥 Target Environment
- Target Machine: Metasploitable 2
- Environment: Isolated Virtual Lab
- Operating System: Vulnerable Linux-based system

---

## 🔍 Assessment Process
1. Network Scanning
2. Service Enumeration
3. Vulnerability Identification
4. Exploitation Verification
5. Shell Access Confirmation
6. Impact Analysis
7. Remediation Recommendations

---

## 📸 Included Evidence
The repository contains:
- Scan result screenshots
- Vulnerable service identification
- Exploitation success proof
- Shell access verification
